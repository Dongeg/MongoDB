<h3>创建数据库</h3>
MongoDB没有创建数据库的命令，如果你想创建一个“demo”的数据库，先执行
```
use demo
```
然后进行一些数据操作，如新建一个demo_collection表并插入一条数据，命令如下
```
> use demo
switched to db demo
> db.demo_collection.insert({x:1})
WriteResult({ "nInserted" : 1 })
> show dbs
demo   0.000GB
local  0.000GB
```
执行<code>show dbs</code>命令看到demo数据库已经建立
